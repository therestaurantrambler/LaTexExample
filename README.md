# LaTexExample
Spanning Tree class notes created in LaTeX

These are the files necessary to complie the Spanning Tree lecture notes document. The SpanningTree.tex file contains the code necessary
to compile the document. (I use WinShell.) The ppdf file is the output. The document uses predefined library commands, as well as, 
specialized definitions. The document contains enumeration, multicolumn, images and tabuluar examples.
